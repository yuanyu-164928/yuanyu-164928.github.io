Sector 01 太阳能发电厂 Boss: SectorGuard

▸
主要目标恢复基地电力 → 重启主电源终端 → 到达顶层 → 击败 SectorGuard
关键区域发电机入口 (Block 01) → 配电中心 (Block 02) → The Concourse (Block 03)
收集品Safe Box ×3, Pure Lunum ×1, Mod ×1, REM ×2, Cartridge Holder ×1, Storage Expander ×1, Mini Cabin Figure ×3
预计时间~1.5–2.5 小时（100% 收集品）

攻略要点

  1. 进入发电机入口，沿路学习基础操作：移动、骇入教程、基础射击
  2. 到达配电中心，面对 5 锁大门。跳到大门的左侧，乘电梯下去拿 Safe Box #1
  3. 在 5 锁大门走廊入口抬头，骇入高处的终端以禁用所有激光障碍。跳下原路障区，清怪后拿 Safe Box #2
  4. 解锁全部 5 个电子锁，进入 The Concourse
  5. 在 Concourse 触发 SectorGuard Boss 战（详细打法见 Boss 攻略页）
回溯提示：Safe Box #3 和 Pure Lunum 需要从 Sector 2 获得的 Lim Eraser 才能返回收集。在配电中心乘电梯上去 → 跳过左侧栏杆 → 用 Lim Eraser 清除蓝色 Lunafilament 封印门 → 击败多波敌人后收集。

Sector 02 量产阵列 Boss: Creator

▸
主要目标乘坐电车 → 骇入 5 个信标 → 解锁巨型大门 → 击败 Creator
关键区域测试场入口 → 购物区 → 连接通道 → 商业区 → 回收控制中心 → 通讯塔广场
收集品Safe Box ×3, Pure Lunum ×2, Mod ×2, REM ×3, Cartridge Holder ×1, Mini Cabin Figure ×3
新能力Lim Eraser（Block 05 解锁）— 清除蓝色 Lunafilament 障碍

攻略要点

  1. 乘坐电车到"假纽约"区，在时代广场区域骇入第一个信标
  2. 穿过购物区和商业区，骇入全部 5 个信标。利用屋顶和小巷躲避巡逻敌人
  3. 在 Block 05 回收控制中心获得 Lim Eraser——核心回溯工具
  4. 所有信标骇入后巨型大门解锁——进入通讯塔广场触发 Creator Boss 战
建议：获得 Lim Eraser 后立即返回 Sector 1 收集之前无法获取的 Safe Box #3。确保 Boss 战前备足 Repair Canister。

Sector 03 Terra Dome Boss: Garden Keeper

▸
主要目标乘坐电车 → 寻找 Eight → 重启 2 个 Lim Regulator → 前往地下重启 Lim Regulator → 击败 Garden Keeper
关键区域Terra Dome 入口 → 生态建模实验室 → 中央大厅 → 土壤研究翼 → Terrarium
收集品Safe Box ×3, Pure Lunum ×2, Mod ×3, REM ×4, Training Data ×2, Mini Cabin Figure ×3

攻略要点

  1. 到达生态建模实验室，重启第一个 Lim Regulator。注意：植物型敌人弱火属性武器
  2. 前往植物工厂重启第二个 Lim Regulator
  3. 剧情遭遇 Eight 后，前往地下 Terrarium 区域
  4. 重启地下 Lim Regulator，触发 Garden Keeper（巨型机器人）Boss 战

Sector 04 Lunum 矿区 Boss: Air Raider → LunaDigger

▸
主要目标乘坐电车 → 重启 IDUS → 前往 Nexus Tower → 潜入仓库 → 击败 Air Raider → 穿越废墟 → 攻入 Nexus Tower → 击败 LunaDigger
关键区域矿区入口 → 吊车操作场 → 仓库 → 主挖掘场 → Nexus Tower
收集品Safe Box ×3, Mod ×3, REM ×3, Pure Lunum ×2, Training Data ×2, Storage Expander ×2, Mini Cabin Figure ×3

攻略要点

  1. 矿区地形开阔——充分利用吊车和采矿设备作为移动平台穿越广阔空间
  2. 潜入仓库触发 Air Raider 战——利用采矿钻机移动，标准骇入-伤害循环
  3. 穿越废墟区域，准备迎战本区最终 Boss LunaDigger
⚠ 警告：LunaDigger 是中期难度门槛。第一阶段骇入窗口极短（Auto Hack 几乎是必需的）。第二阶段可骇入小型无人机反制 Boss。详见 Boss 攻略页。

Sector 05 实验性 Pragmatics 线性 · 无收集品

▸
主要目标前往实验区 → 营救 Diana → 获取能量反应堆 → 护送 Diana 返回 Shelter
新能力Cleanse — 按住 LB/E 充能释放，驱散红色 Lunafilament 障碍
重要本区 不可 通过快速旅行返回。

攻略要点

  1. 这是纯线性叙事区域，无收集品——聚焦于剧情推进和情感冲击
  2. Diana 暂时离队，Hugh 需要独自战斗。敌人密度高——谨慎管理弹药
  3. 获得 Cleanse 能力后可清除红色 Lunafilament，这是解锁多个前序区域隐藏房间的关键
  4. 护送 Diana 阶段优先消灭远程敌人
⚠ 离开前注意：本区无遗漏内容，但获得 Cleanse 后立刻回溯前序区域清除红色 Lunafilament，收集遗漏物品。

Sector 06 中央港 Boss: Sentinel → Eight/Idea → Abiosis

▸
主要目标乘坐电车 → 与 Eight 汇合 → 抵达核心区 → 前往轨道电梯 → 净化 Lunafilament 源头 → 骇入 2 个终端 → 三连 Boss 战
关键区域港口入口 → 研究翼 → Regolith 实验室 → Lunafilament 实验室 → 轨道电梯
收集品Safe Box ×3, Mod ×3, REM ×3, Training Data ×2, Pure Lunum ×2, Storage Expander ×1, Mini Cabin Figure ×3

攻略要点

  1. 这是最后一关——确保所有武器升到满级，Repair Canister 备足
  2. 在 Lunafilament 实验室使用 Cleanse 净化源头，然后骇入两侧的终端
  3. 最终连战包括：Sentinel → Eight/Idea → Abiosis——中间没有存档点
  4. 击败 Abiosis 观赏默认结局
真结局提示：要解锁隐藏真结局，需先在"Unknown Signal"模式中获得 Black Box Mod，装备后再次击败最终 Boss。详见 Tips & Endgame 页面。
下一区: Epilogue ▸ ▲ 回顶部

Epilogue 最终决战

▸

使用 Eraser 击败 Abiosis 的最终形态，跟随 Diana 完成游戏。尾声为纯过场动画，无收集品。

通关后解锁：Unknown Signal 模式、New Game+、章节选择（回溯收集品）。
